# Gericht Restaurant Landing

Premium restaurant landing page built with React.

## Run locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy to Vercel

This project is configured for static output via CRA.

- Build command: `npm run build`
- Output directory: `build`
- Optional config is in `vercel.json`

See `DEPLOY_VERCEL_CHECKLIST.md` for a full deployment checklist.
