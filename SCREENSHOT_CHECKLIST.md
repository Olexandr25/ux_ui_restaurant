# Screenshot Checklist (Fiverr Portfolio)

## Required shots

1. Full desktop hero (with headline + CTA visible)
2. Full desktop mid-page (menu + visual assets)
3. Desktop trust block (awards + testimonials)
4. Desktop contact/booking section
5. Tablet viewport screenshot
6. Mobile viewport screenshot
7. Gallery close-up
8. Footer + newsletter close-up

## Capture settings

- Desktop: 1440x900
- Tablet: 834x1112
- Mobile: 390x844
- Browser zoom: 100%
- Hide bookmarks bar and extensions

## Quality rules

- Keep text sharp and readable
- Do not crop CTA buttons out
- Avoid unfinished placeholder content in frame
- Keep one consistent browser theme across all screenshots

## Optional bonus assets

- 15-25 second scroll video
- Before/after comparison image
- 1 cover image with project title overlay
